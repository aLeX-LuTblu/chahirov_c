Выполнил задачи для лабораторной работы №3. Использовал библиотеки stdio.h, stdlid. ответы на вопросы: 1)Что делает malloc Выделяет блок сырой памяти заданного размера (в байтах) в куче. 2)Чем отличается malloc от calloc malloc оставляет в памяти «мусор», а calloc обнуляет все выделенные байты. 3)Что делает realloc Изменяет размер ранее выделенного блока памяти, стараясь сохранить текущие данные. 4)Что произойдет при утечке памяти Память не возвращается системе после завершения работы функции. Программа постепенно заполняет ОЗУ. 5)Зачем нужен free Чтобы пометить выделенный блок памяти как свободный, позволяя системе или другим частям программы использовать его повторно. 6)Что произойдет при двойном освобождении памяти Возникнет ошибка выполнения, программа аварийно завершится. 7)Как проверить, что malloc вернул NULL Использовать условие: if (ptr == NULL) {}. 8)В чем отличие статического и динамического массива Статический: размер задается до запуска программы. Динамический: размер задается во время работы программы.
