# Лабораторная работа №1: Типы данных, память и выражения в С

## Описание
в данной лабораторной работе изучаются базовые типы данных языка C (int, float, double, char), оператор sizeof, работа с адресами памяти, арифметические операции, преобразование типов, переполнение и потеря точности.

## Технологии
- Язык программирования: C
- Компилятор: GCC (MinGW для Windows)
- Система контроля версий: Git
